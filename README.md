## Featured Project

### 🚌 Korat Songthaew Passenger Tracker

เว็บแอปสำหรับผู้โดยสารรถสองแถวจังหวัดนครราชสีมา รองรับการติดตามรถออนไลน์ ดูเส้นทาง เลือกจุดรับ ส่งคำขอเรียกรถ และติดตามเวลาที่รถจะมาถึง

- Live vehicle locations with Supabase Realtime
- Interactive routes and stops with Leaflet and OpenStreetMap
- Passenger GPS, automatic pickup selection, and ETA
- Responsive experience for phones, tablets, and iPad
- Request lifecycle shared with a separate driver application
